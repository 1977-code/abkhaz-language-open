# Черновик главы L04: День и действия

Статус: skeleton draft. Глагольные модели требуют научной редакции.

## Что вы научитесь делать

- назвать простые ежедневные действия;
- спросить, что делает другой человек;
- рассказать коротко о своем дне.

## 1. Сценарий

Два человека говорят о том, что они делают утром, днем и вечером.

## 2. Диалог-функция

```text
A: Что ты делаешь утром?
B: Утром я ...
A: А вечером?
B: Вечером я ...
```

Абхазская версия: `needs_native_review`.

## 3. Глагольная рамка

В этой главе используются только 5-8 глаголов. Никакой полной таблицы глагола.

Для каждого глагола:

- форма для "я";
- форма для "ты/вы" только если нужна в диалоге;
- форма для "он/она" только если нужна в упражнении;
- аудио;
- 3 drills.

## 4. Упражнения

- сопоставьте действие и картинку;
- замените участника;
- ответьте по образцу;
- восстановите порядок дня;
- расскажите о себе по карточкам.

## 5. Контроль

5-7 фраз о своем дне.
