# Editorial pipeline

## Поток производства главы

1. Method brief.
2. Russian function draft.
3. Lexical list.
4. Grammar scope.
5. Native phrase creation.
6. Scientific review.
7. Exercise draft.
8. Audio script.
9. Recording.
10. Pilot.
11. Revision.
12. Publish.

## Статусы главы

| Статус | Значение |
|---|---|
| `brief` | есть методический бриф |
| `function_draft` | есть русская функция и структура |
| `native_review` | абхазские фразы у носителя |
| `scientific_review` | грамматика у научного редактора |
| `audio_ready` | текст готов к записи |
| `recorded` | аудио записано |
| `pilot` | глава тестируется |
| `revised` | внесены правки после пилота |
| `published` | можно публиковать |

## Кто принимает

| Блок | Принимает |
|---|---|
| Методика | методист |
| Норма | научный редактор |
| Естественность | редактор-носитель |
| Аудио | аудиопродюсер + носитель |
| Права | менеджер по правам |
| UX | разработчик/дизайнер |
