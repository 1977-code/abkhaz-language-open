# Политика лицензирования контента

Статус: рабочая версия. Перед публичным запуском нужна юридическая проверка.

## Цель

Сделать новые материалы проекта максимально полезными и переиспользуемыми, не нарушая права сторонних авторов.

## Рекомендуемая модель

| Материал | Рекомендуемый статус |
|---|---|
| Авторский текст уроков | CC BY 4.0 |
| Упражнения и ключи | CC BY 4.0 |
| Таблицы, схемы, объяснения | CC BY 4.0 |
| Частотный список, если он не содержит ограниченных источников | CC0 |
| Метаданные уроков | CC0 |
| Собственные учебные примеры | CC BY 4.0 или CC0 по решению |
| Аудио дикторов | отдельная лицензия по speaker release |
| Материалы партнеров | строго по договору |

## Нельзя

- Публиковать чужие PDF как часть курса без разрешения.
- Переписывать учебник близко к исходному тексту и выдавать за новый.
- Смешивать открытые данные с `research-only` без маркировки.
- Обучать AI на спорных материалах.
- Использовать голос диктора шире, чем разрешено в релизе.

## Атрибуция

Для каждого стороннего материала хранить:

```yaml
source_id:
title:
author:
url:
license:
retrieved_at:
allowed_use:
attribution_text:
notes:
```

## Публичная формула для сайта

> Материалы курса, созданные командой проекта, публикуются на условиях открытой лицензии, если рядом с конкретным материалом не указано иное. Сторонние тексты, аудио и изображения используются только по разрешению правообладателей или в рамках явно указанных лицензий.
