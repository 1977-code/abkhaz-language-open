# План корпусного слоя

## Задача

Создать учебный корпус, который помогает:

- выбирать частотную лексику;
- находить живые примеры;
- строить словарь курса;
- проверять грамматические модели;
- готовить AI-тьютора.

## Стартовые источники

| Источник | Роль | Статус |
|---|---|---|
| Apsua.online | основной корпусный ориентир, частотность, морфология | нужен запрос условий |
| CLARINO Abkhaz National Corpus | открытый seed | проверить и скачать доступный слой |
| Common Voice transcripts | короткие фразы и произношение | использовать по правилам |
| Собственные тексты курса | правочистое ядро | создавать с нуля |
| Партнерские тексты | расширение | только после разрешений |

## Первые задачи этапа 2

- Получить или вручную собрать частотный список из доступных корпусных источников.
- Разделить лексический минимум на `A0`, `A1`, `A2`.
- Составить список 100-150 разговорных формул.
- Подготовить формат карточки слова.
- Подготовить формат карточки примера.
