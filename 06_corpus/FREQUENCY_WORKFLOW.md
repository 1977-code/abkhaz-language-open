# Workflow частотного словаря

## Цель

Получить учебный частотный минимум `A0-A2`, который не является случайным списком слов.

## Источники

| Слой | Статус |
|---|---|
| CLARINO / Abkhaz National Corpus | top-5000 лемм получен, очередь 1500 кандидатов готова к проверке |
| Apsua.online | нужен запрос условий |
| Common Voice Abkhaz 25.0 | импорт готов, архив требует принятия условий платформы |
| Собственные тексты курса | own-original |
| Партнерские тексты | только после разрешений |

## Шаги

1. Получить open corpus seed. **Готово для ABNC.**
2. Очистить очевидный технический мусор. **Готова автоматическая очередь v1.**
3. Нормализовать формы, где возможно.
4. Сгруппировать леммы.
5. Отдельно выделить служебные слова и формулы.
6. Сопоставить частотность с главами.
7. Ручная проверка лексикографом.
8. Проверка носителем на естественность и регистр.
9. Разметка `A0`, `A1`, `A2`.
10. Экспорт в `course_lexicon`.

## Выходной формат

```csv
lemma_ab,ru,level,frequency_band,domain,chapter,active_or_passive,source_id,rights_status,native_review
```

## Важное правило

Частотность не равна учебной нужности. Слово включается в главу только если оно нужно для сценария или является системно важным.

Текущее состояние и ограничения ABNC описаны в `06_corpus/ABNC_FREQUENCY_V1.md`.
