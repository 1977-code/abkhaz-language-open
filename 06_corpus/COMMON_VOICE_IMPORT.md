## Ограничения доступа и хранения

- Для скачивания нужен аккаунт Mozilla Data Collective, принятие условий набора и API-ключ либо загрузка через веб-интерфейс.
- API-ключ проверен 11 июня 2026 года; запрос download session вернул `403 Terms must be accepted`. После ручного принятия условий тот же запрос можно повторить без изменения скрипта.
- Нельзя пытаться установить личность дикторов.
- Нельзя повторно размещать или распространять исходный архив.
- В git не попадают MP3, исходные TSV, `client_id`, пути клипов и демографические поля.
- В проекте хранится только агрегированный частотный CSV без предложений и идентификаторов.

## Воспроизводимый импорт

После получения официального архива поместить его в игнорируемую git папку `06_corpus/local-source-data/` и запустить:

```bash
python3 06_corpus/scripts/freq_from_common_voice.py \
  06_corpus/local-source-data/common-voice-abkhaz.tar.gz \
  data/freq_common_voice_ab_v0.csv \
  3000
```

Скрипт:

1. читает `validated.tsv` прямо из архива, не распаковывая аудио;
2. поддерживает поля `sentence` и `text` разных выпусков;
3. удаляет повторяющиеся промпты, чтобы количество дикторов не искажало частотность;
4. не экспортирует исходные предложения;
5. считает частоту словоформ и долю уникальных предложений, где встретилась форма.

## Выполнено 11.06.2026: срез по открытому пулу предложений

Официальный архив (4,76 GiB) лежит за регистрацией Mozilla Data Collective, поэтому первый фактический срез построен по **открытому пулу предложений** локали `ab` из репозитория common-voice/common-voice (server/data/ab, лицензия предложений CC0) — без регистрации и принятия условий.

- скрипт: `06_corpus/scripts/freq_from_cv_sentence_pool.py`;
- результат: `data/freq_cv_pool_v0.csv` (топ-5000 словоформ);
- объём: 1 046 762 уникальных предложения, 6 567 225 токенов, 924 658 уникальных словоформ;
- топ листа разговорный (местоимения, союзы, частицы: `уи`, `ҳәа`, `аха`, `сара`, `иара`…) — в отличие от энциклопедического перекоса Википедии;
- это частоты текстового пула для записи, не записанной речи; срез по валидированным клипам остаётся за `freq_from_common_voice.py` после получения архива.

## Как использовать результат

1. Сопоставить с `data/freq_abwiki_v0.csv`.
2. Отделить универсальные формы от энциклопедического и дикторского перекоса.
3. Не называть список лемматизированным: Common Voice даёт словоформы.
4. Сверить кандидатов с частотностью ABNC и syllabus A0–A2.
5. Любую учебную фразу отдельно проверить носителем.
