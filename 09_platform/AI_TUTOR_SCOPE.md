# AI-тьютор: рамка первой версии

## Принцип

AI-тьютор запускается только поверх проверенных материалов курса. Он не должен свободно генерировать абхазский текст без маркировки и проверки.

## Разрешенные функции MVP

| Функция | Описание | Источник |
|---|---|---|
| Объяснить тему | пересказывает грамматику конкретной главы | утвержденные уроки |
| Проверить упражнение | сверяет ответ с answer key | упражнения |
| Controlled dialogue | ведет диалог только по разрешенным репликам | dialogue bank |
| Подсказать слово | показывает слово из словаря курса | course lexicon |
| Показать пример | берет пример из правочистого корпуса | approved examples |
| Объяснить ошибку | использует реестр типичных ошибок | error register |

## Запрещенные функции MVP

- свободный перевод любых русских фраз на абхазский;
- генерация новых абхазских диалогов без проверки;
- использование `research-only` источников;
- ответы о диалектах без утвержденной базы;
- имитация носителя там, где нет уверенности.

## Ответ AI при недостатке данных

> В материалах курса пока нет проверенной формы для этой фразы. Я могу объяснить, какая тема здесь нужна, и отметить это как запрос на проверку носителем.

## Минимальная база перед запуском

- 10-12 опубликованных глав;
- утвержденный словарь курса;
- approved examples;
- answer keys;
- error register;
- rights-clean corpus subset;
- запрет на доступ к `research-only` материалам.
