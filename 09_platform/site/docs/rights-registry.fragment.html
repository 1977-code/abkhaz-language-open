Реестр прав и лицензий

Цель реестра — не допустить смешения юридически чистых материалов с материалами, доступными только для исследования.

Статусы

Статус Значение Использование
public-open есть явная открытая лицензия или публичный статус можно использовать по условиям лицензии
licensed-third-party есть/будет отдельное письменное разрешение можно использовать в рамках договора
research-only можно смотреть и анализировать, но нельзя публиковать/перепаковывать только внутренние заметки
permission-pending нужен запрос правообладателю не публиковать до ответа
do-not-use-until-cleared статус неясен или конфликтен не использовать в продукте
own-original создано командой проекта можно публиковать по выбранной лицензии

Рекомендуемая лицензионная модель для нового контента

Тип материала Рекомендуемая лицензия
Авторский текст учебника, упражнения, таблицы CC BY 4.0
Лексические базы, частотники, метаданные, sentence alignment CC0, если нет ограничений от источников
Аудио носителей отдельное согласие; публичная лицензия выбирается договором
Иллюстрации и дизайн CC BY 4.0 или отдельная коммерческая лицензия
Материалы партнеров строго по договору
