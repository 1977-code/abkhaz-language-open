# Сайт abkhlang.ru (PHP, динамический)

Двуязычная (RU/EN) витрина проекта. Сервер: Caddy + PHP 8.3-FPM, автоматический HTTPS
(Let's Encrypt). Никаких фреймворков — общие шаблоны через `inc/`.

## Стек
- PHP 8.3 (общие шаблоны через `inc/`, без фреймворка)
- Caddy (роутинг, чистые URL документов, заголовки безопасности)
- Форма обратной связи `contact.php` (PHP mail + согласие на обработку ПДн по 152-ФЗ)

## Структура
- `index.php`, `en/index.php` — главные RU/EN
- `inc/` — config, functions, head, header, footer, contact_form (служебное, закрыто на уровне веб-сервера)
- `docs/index.php` — рендер документа из `docs/<slug>.fragment.html` в общий шаблон
- `docs/*.md` → `docs/*.fragment.html` (сборка `./build.sh`, marked)
- `contact.php` — обработчик формы (PRG + honeypot)
- `assets/`, `data/` — статика и открытые CSV
- `404.php` — тематическая страница

## Локальный запуск
```bash
./build.sh                       # собрать docs/*.md → docs/*.fragment.html
php -S localhost:8000            # локальный просмотр
```

## Деплой и конфигурация
Деплой — сборкой (`build.sh`) и синхронизацией на сервер (rsync/SSH). Хост-специфичные
значения (домен, адрес получателя формы, путь журнала обращений) задаются в
`inc/config.php`; для приватных значений используйте локальную, не коммитимую копию
(`inc/config.local.php`) — она в `.gitignore`. Реальные адреса сервера, ключи и DNS
в этот репозиторий намеренно не включены.

## Почта формы
Если на сервере нет MTA, `mail()` не доставляет — форма показывает запасной вариант
(«напишите на почту»). Для доставки настройте SMTP-релай (например, msmtp).
