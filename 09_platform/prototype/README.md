# Статическая витрина проекта

Публичный адрес: https://abkh.vercel.app (Vercel-проект `abkh`; старый адрес prototype-three-bice.vercel.app сохранён как дополнительный домен).

Локально: открыть `09_platform/prototype/index.html`, сервер не нужен.

Это не финальный сайт курса, а публичная витрина для партнёров. В `docs/` публикуются только методические документы; внутренние реестры, контакты, бюджет и операционные отчёты сюда не копировать.

## Что показывает

- цель проекта;
- статус этапов;
- состав MVP;
- дорожную карту;
- ссылки на главные документы;
- пакеты для носителя, научного редактора, пилота и записи.

## Ограничения

- нет реальных уроков;
- нет аудио;
- нет проверенных абхазских фраз;
- документы публикуются как стилизованные HTML-страницы: после правок в `docs/*.md` запустить `./build-docs.sh` (нужен `npx marked`), затем `vercel deploy --prod --yes`; исходные `.md` тоже отдаются (как `text/plain` через `vercel.json`).
