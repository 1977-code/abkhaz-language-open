# Style guide курса

## Язык объяснений

Объяснения пишутся для русскоговорящего взрослого новичка.

Принципы:

- короткие предложения;
- один новый грамматический принцип за раз;
- термин вводится только там, где он помогает;
- сначала пример, потом правило;
- сложные исключения выносятся в справку;
- не использовать академическую формулировку без перевода на учебный язык.

## Маркировка форм

| Метка | Значение |
|---|---|
| `норма` | форма основного курса |
| `разговорное` | часто встречается в речи, но требует комментария |
| `диалектное` | вариант, связанный с диалектом |
| `книжное` | встречается в текстах, но не является первым выбором новичка |
| `B1+` | важно, но рано для текущего уровня |

## Абхазские примеры

Для каждого примера желательно хранить:

- абхазскую форму;
- транслитерацию или чтение, если нужно на раннем этапе;
- дословный разбор;
- естественный русский перевод;
- аудио ID;
- источник;
- статус прав.

## Нельзя

- Смешивать диалекты без маркировки.
- Давать пять исключений до первого живого примера.
- Копировать формулировки из академических источников в учебник.
- Объяснять глагольную систему целиком в первой главе.
