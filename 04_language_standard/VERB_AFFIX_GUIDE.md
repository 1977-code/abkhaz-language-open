## 1. Типология служебных морфем (по Джонуа, введение)

- Классы: деривационные (превербы), реляционные (личные префиксы, временные суффиксы), деривационно-реляционные (каузатив р-, корне-суффиксы).
- Позиции: префиксы · **превербы** (особый класс: в переходных глаголах между превербом и корнем **вклиниваются классно-личные аффиксы**: и-қә-и-ргылеит «он поставил это на что-то») · инфиксы (редки) · суффиксы · **корне-суффиксы** (четвёрка -ла, -ҵ, -ҵа, -х — бывшие корни) · циркумфиксы (ал…к) · **амбификсы** (отрицание м-: в статических всегда суффикс, в динамических префикс или суффикс).
- По продуктивности: продуктивные · непродуктивные · унификсы (один преверб — одно слово: ӡыр- только в аӡыргара «прославлять»).
- Нулевой аффикс: при стечении двух и- один выпадает (и-с-ҭеит < и-ø-с-ҭеит).

## 2. Слотовая модель формы (для чтения формы слева направо)

Полная линейная схема (упрощена; сверить с научредом):

| № | Слот | Примеры |
|---|---|---|
| 1 | абсолютивный показатель (Column I): кто/что «движется/есть» | д- (человек), и- (вещь), с-, у-, б-, ҳ-, шә-, р- |
| 2 | направительный/локальный преверб | аа- «сюда», на- «туда»; ҭа-, қә-, ҩна-, шьҭа-, д-… |
| 3 | косвенный объект + послеложный элемент (Column II) | и-, л-, а-, з- «для»… |
| 4 | эргативный агент (переходные; ближе всех к корню) | и-и-ҭеит «он дал это ему» |
| 5 | отрицание м- (динамические, префиксальная позиция) | иҟаи**м**ҵеит |
| 6 | каузатив р- | и-қә-и-**р**-гылеит |
| 7 | **КОРЕНЬ** (± корне-суффикс -ла/-ҵ/-ҵа/-х) | |
| 8 | аспектуально-модальные суффиксы | -ла, -х, -цәа, -ӡа… (см. §5) |
| 9 | время/наклонение | -ует/-оит, -ит, -он, -н, -п, -шт, -зар… |
| 10 | отрицание -м (суффиксальная позиция) / вопросительные | -ӡам, -ма, -и/-зеи, -да |

Учебная редукция для A0–A2: **три окна** — «кто» (слот 1) · «действие» (корень) · «когда» (слот 9); с главы 4 добавляем окно «кому/куда» (слоты 2–4). Полная схема — только в справочнике курса (Grammatical Summary), не в основном тексте.

## 3. Времена и наклонения (архитектура по Чкадуа-1970)

Организация системы — **три серии основ**: чистой основы · настоящего времени · аориста. Внутри серий:

- **Статические глаголы** — 2 формы: настоящее (-уп) и прошедшее неопределённое (-н) + желательное, условное, допускательное, заглазные.
- **Динамические глаголы** — изъявительное: настоящее · прошедшее несовершенное (имперфект) · аорист · прошедшее неопределённое (перфект) · давнопрошедшее · давнопрошедшее упреждающее · будущее определённое (I) · будущее неопределённое (II).
- **Наклонения**: условное (богатый набор суффиксов: -р/-ри, -зар/-зари/-зтән, -уазар, -хазар, -разар, -зҭгьы), сослагательное I/II, желательное, повелительное, **допускательное** (-ааит: игааит «пусть возьмёт» — школьный учебник 6 кл. даёт его как «желательную форму»), **заглазные I/II** (-заап/-заарын «оказывается» — эвиденциальность).
- **Инфинитные образования**: инфинитивы 4 семантик, деепричастия, вопросительные формы (отдельное спряжение!), отрицательные формы.
- **Способы действия** (гл. III): многократный, чрезмерность, интенсивный, повторный, начинательный, дюративный, ограничительный, смягчительный — морфологический аспект, см. §5.

Разметка по уровням курса: **A0–A2** — статические 2 формы; динамические: настоящее, аорист, будущее I (+ имперфект и перфект рецептивно, гл. 9–12); повелительное (м/ж!), базовое условное -зар, допускательное -ааит как формула («Аныҳәа удыл**ааит**!» — из благопожеланий). **B1-мостик** — остальные времена, сослагательные, заглазные, полный вопросительный ряд.

## 4. Отрицание (амбификс м-)

- Статические: суффикс — иҟоуп → иҟа**ӡам**; сыҟоуп → сыҟа**м**.
- Динамические: **финитные формы — суффикс** (сцоит → сц**ом**, дцалоит → дцал**ом**), **инфинитные и императив/аорист — префикс** (сы**м**цакәа «я не идя», у**м**ца «не ходи!», иҟаи**м**ҵеит «он не сделал»).
- Для курса: даём как «два дома отрицания» с опорными парами; полная таблица позиций — в справочник.

## 5. Аспектуально-модальные суффиксы (по Авидзба З., общие для всех диалектов)

Дидактическое золото для русскоговорящих: **русские наречия/частицы «живут внутри абхазского глагола»**.

| Суффикс | Значение | Русское соответствие | Пример (внутр.) |
|---|---|---|---|
| -ла | многократность | «обычно, много раз» | дыҩналалон «каждый раз заходил» |
| -х | повторность/возврат | «опять; уже; больше (не)» | иҟасҵа**х**т «я опять сделал» |
| -қәа | множественность действия | «неоднократно» | сца**қәа**уон |
| -цәа | чрезмерность | «слишком» | дца**цәа**ит |
| -ӡа | интенсивность | «очень, совсем; даже» | дгәаа**ӡ**ит |
| -ҵәҟьа | достоверность | «действительно, именно, как раз» | сынха**ҵәҟьа**ит |
| -цыԥхьаӡа | итеративная связь | «каждый раз, как» | иҳәа**цыԥхьаӡа** |
| -ижьҭеи | точка отсчёта | «с тех пор, как» | дце**ижьҭеи** |
| -наҵы / -аанӡа | предел | «пока; до тех пор, пока» | утәа**наҵы**; дца**анӡа** |
| -шәа | ирреальное сравнение | «как будто» | |
| -ишьт | смягчённое побуждение | «-ка» | утәе**ишьт** «сядь-ка» |
| -гьы / агь- | присоединение | «и, тоже» | сцар**гьы** «если я и пойду» |
| еиҭа- | повторность (префикс) | «опять, снова» | д**еиҭа**цеит |

## 6. Локальные превербы + корне-суффиксы (по Авидзба А. и Джонуа)

Инвентарь: **>110 локальных превербов**; приложения Авидзбы дают готовые списки: с -ла — 42, с -ҵ — 36, с -ҵа — 44, с -х — 41.

Педагогическое ядро — **матрица 2×2** «направление × переходность» на корне-суффиксах:

| | внутрь/на место | наружу/с места |
|---|---|---|
| **непереходный** («сам») | -ла: аҭа**ла**ра «войти» | -ҵ: аҭы**ҵ**ра «выйти» |
| **переходный** («что-то») | -ҵа: аҭа**ҵа**ра «положить внутрь» | -х: аҭы**х**ра «вынуть» |

Одна четвёрка на превербе ҭа- «внутри» даёт русскоговорящему сразу «войти/выйти/положить/вынуть» — кандидат в объяснение-озарение главы о превербах. Аналогично: ҽыжә- (сесть на коня/спешиться/посадить/снять), шьа- (обуть/разуть), шә- (одеть/снять одежду), гара- (люлька: положить/вынуть).

- **Аблаут гласного преверба**: исход -а ↔ -ы/ø различает направление (гара-ҵа-ра «положить В люльку» ↔ гар-х-ра «вынуть ИЗ люльки»). В терминах источников есть разнобой (у Джонуа во введении «а → элативное (центростремительное)», а его же пример гара- и вся система Авидзбы дают обратное) — **в курсе не используем термины иллатив/элатив вовсе**, только «внутрь/наружу»; вопрос терминов — научреду.
- **Направительные суффиксы -аа/-ла** чинят «дефект» превербов без гласного: дықәлеит «поднялся» → д**аа**қәлеит «поднялся сюда» (тема Клычева — дополнить после конвертации djvu).
- Десемантизация направительных превербов аа-/на-/ҩа-/ла- → «характер действия» (внезапность, однократность): даатәеит ≈ днатәеит ≈ длатәеит «присел» (Джонуа, ст. аа4).
- Превербы частей тела и места (гәы- «сердце/грудь», ҿа- «рот», шьа- «нога», ҩна- «дом», ӡаа- «вода») — мостик «существительное → преверб» полезен для запоминания.
- Учебный набор превербов A2 (черновой, утвердить): ҭа- (в), қә- (на), ла- (в массу/жидкость), д- (к, присоединение), ҩна- (в дом), аа-/на- (сюда/туда), шьҭа- (вслед/лечь), кыд- (на стену/висящее). Остальные — словарно, по мере текстов.

## 7. Дидактическая линия «как объясняем глагол» (сводка для брифов)

1. Гл. 1–3: формы-гештальты (сцоит, иҟоуп) без разбора; только «кто» — первая буква.
2. Гл. 4 (первый глагольный узел): три окна «кто—действие—когда»; статические vs динамические; отрицание «два дома».
3. Гл. 5–8: окно «кому/куда»; повелительное м/ж; аорист; будущее I; -гьы, -ӡа.
4. Гл. 9–12: матрица превербов 2×2 (одно «озарение», не список 110); имперфект/перфект рецептивно; -х, еиҭа-; допускательное как формула благопожеланий.
5. Справочник (Grammatical Summary): полная слотовая схема §2, таблицы §3–6, отсылки к Джонуа/Чкадуа для любопытных.
