# Критерии приемки урока

Урок нельзя считать готовым, пока он не проходит эти проверки.

## Контент

- Есть речевая цель.
- Есть диалог или функциональный сценарий.
- Есть словарь главы.
- Есть 1 главный грамматический фокус.
- Есть блок фонетики.
- Есть упражнения на узнавание, понимание и производство.
- Есть повторение прошлых глав.
- Есть контрольное задание.

## Методика

- Ученик понимает, что делать без преподавателя.
- Новая грамматика не появляется без примеров.
- В главе нет больше одной тяжелой новой идеи.
- Сложность растет постепенно.
- Есть блок "ошибки русскоговорящих".

## Язык

- Все абхазские фразы проверены носителем.
- Спорные формы имеют помету.
- Диалектные формы не смешаны с основной нормой.
- Грамматические объяснения проверены научным редактором.

## Права

- У каждого внешнего примера есть источник.
- У каждого источника есть `rights_status`.
- Нет материалов из `research-only` в публичном тексте.
- Аудио имеет `speaker_release_id`.

## Техника

- Урок имеет `lesson_id`.
- Все слова имеют `word_id`.
- Все аудио имеют `audio_id`.
- Все упражнения имеют ключи.
- Материал можно экспортировать на сайт.
