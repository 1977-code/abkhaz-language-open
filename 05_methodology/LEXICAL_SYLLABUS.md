# Лексический syllabus

Статус: 2026-07-02 — выполнен анализ №7 (словарь Микаиа ↔ частотное ядро ABNC v2 ↔ NorthEuraLex ↔ Викисловарь); метод отбора зафиксирован, рабочий стол `data/lexical_core_workbench.csv` готов. Уровни и переводы ещё не назначены.

## Принцип

Слова отбираются по четырём фильтрам:

1. частотность (ядро ABNC);
2. нужность для сценария главы;
3. **учебная традиция** — слово подтверждено школьным ядром (Микаиа-1985, словники школьных учебников, Кварчелия);
4. простота фонетического и грамматического ввода (порядок внутри главы).

Слово входит в главу, если его требует сцена **и** оно проходит фильтр 1 или 3. Системные множества (местоимения, числительные обоих классов, вопросительные слова) входят целиком по грамматическому плану независимо от частоты.

Корпусная очередь: `data/lexical_review_queue_v1.csv` (срез v1 с морфемной сегментацией и колонками ревью). Значения `corpus_lemma` нельзя автоматически переносить в карточки: морфемную сегментацию, ударение и словарную форму должен подтвердить лексикограф.

## Анализ №7 (02.07): что говорят данные

### Частотное ядро ABNC v2

Официальная POS-выгрузка Меурера (`data/freq_abnc_lemma_pos_v2.csv`, топ-6000) содержала дубли по регистру (уи/Уи) — после слияния **5377 лемм**. Покрытие письменного корпуса: топ-100 → 35,6% · топ-500 → 59,4% · топ-1000 → 70,7% · топ-2000 → 82,4% · топ-3000 → 89,6%. Состав топ-2000: 45% сущ., 27% глаг., 11% нареч., 5% прил., 4% местоим., остальное — служебные. Топ-100 (треть любого текста) — служебный каркас: он распределяется по курсу как грамматический материал узлов, а не как «словарные слова».

Оговорки: (1) лемматизация глаголов неполная — в топе как «леммы» сидят финитные формы (иҳәеит, ауп, акәын), глагольные гнёзда собираем вручную; (2) неоднозначные разборы взвешены 1/nm (предупреждение Меурера); (3) **письменный корпус систематически недооценивает бытовую лексику**: из 1125 форм базового списка NorthEuraLex **745 концептов отсутствуют в топ-6000** — включая «глаз», «зуб», «шея». Для разговорного A0–A2 частотность как единственный критерий непригодна — отсюда обязательность фильтров 2–3.

### Словарь Микаиа (1985) — педагогическое сито

«Аԥсуа бызшәа арҵагатә жәар» (Аҟәа: Алашара, ~1300 слов; научред В. А. Касландзиа; скан 139 с. — `source_files/dictionaries/2880-...`). Первый абхазский толковый **учебный** словарь: слова отобраны методистом из школьных учебников 1–10 классов, толкования по-абхазски. Анатомия статьи: заглавное слово → **для глаголов: масдар + финитная форма в скобках** (АИБАРГУТОРА (еибаргутоит)) → толкование → живая цитата из литературы с автором (Шинкуба, Гогуа, Папаскир…) → фразеология под ромбом.

Берём: цитационную форму глагола «масдар + финитная» (прецедент традиции — решает вопрос словарной формы; согласовать с научредом); принцип «каждое слово в живом примере»; сам словник как фильтр 3 (распознать при сборке словаря). Не берём: монолингвные толкования (наши — русские), отсутствие ударений.

### Викисловарь — почти ничего

Готовый глосс в англ. Викисловаре есть лишь у ~8–10% лемм топ-2000; есть ловушки омонимов (аха 'но', топ-3 корпуса, глоссирован только как 'кол'). Автоматических глоссов не будет: весь учебный словарь глоссируем сами и валидируем носителем. Викисловарь — точечная справка (произношение, этимология).

### Рабочий стол отбора

**`data/lexical_core_workbench.csv`** — топ-2500 слитых лемм v2: rank, lemma, pos, freq, накопленное покрытие, концепт NorthEuraLex, викиглосс, status=needs_native_review. По мере распознания словников добавим колонки «Микаиа», «6 класс», «Кварчелия», «глава». Отношение к `lexical_review_queue_v1.csv`: workbench даёт актуальные ранги/POS (v2), очередь v1 — сегментацию и процесс ревью; при сборке словаря объединить по леммам.
