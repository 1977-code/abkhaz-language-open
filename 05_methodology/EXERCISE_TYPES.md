# Типы упражнений

## Упражнения без риска непроверенных фраз

Эти типы можно проектировать до финальной носительской проверки, подставляя проверенные фразы позже.

| Тип | Назначение | Формат |
|---|---|---|
| Match | связать русскую функцию и абхазскую фразу | пары карточек |
| Listen and choose | распознать услышанное | 3-4 варианта |
| Slot substitution | заменить имя/страну/число | шаблон с одним слотом |
| Dialogue ordering | собрать диалог | перетасовка реплик |
| Minimal pair recognition | услышать различие | A/B выбор |
| Picture prompt | сказать по картинке | controlled production |
| Micro-translation | перевести короткую фразу | обе стороны |
| Error spotting | найти типичную ошибку | 2 варианта |
| Shadowing | повторить ритм | аудио + запись ученика |
| Spaced review | повторить старую лексику | mixed review |
