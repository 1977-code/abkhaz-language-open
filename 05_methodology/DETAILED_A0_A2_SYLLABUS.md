# Подробный syllabus A0-A2

Статус: production draft. Обновлено 2026-07-01 по итогам бенчмарка Кварчелии (`BENCHMARK_KVARCHELIA.md`) — её методика апробирована в АГУ, цель совпадает с нашей (элементарный уровень для начинающих с нуля).

Принцип: каждая глава учит не "тему грамматики", а речевое действие. Грамматика появляется как инструмент для этого действия.

## Микроструктура урока (триада, из бенчмарка Кварчелии)

Внутри каждой главы лексика и упражнения идут по нарастающей трудности:

1. **Ввод** — слова/формулы на запоминание (с картинкой и аудио).
2. **Отработка** — упражнения на закрепление (подстановка, выбор, сборка).
3. **Двусторонний перевод** (рус↔абх) — самый трудный этап, финал главы.

Каждое упражнение помечается уровнем сложности; аудио привязано к вводу и отработке.

## Карта уровней

| Уровень | Что умеет ученик | Объем |
|---|---|---|
| A0 | читает, слышит базовые звуки, узнает формулы, не боится письма | модуль 0 + глава 1 |
| A1.1 | знакомится, говорит о себе, семье, языке, простых действиях | главы 1-4 |
| A1.2 | действует в городе, магазине, быту, времени и сервисе | главы 5-8 |
| A2 | рассказывает простые истории, читает адаптированные тексты, понимает варианты | главы 9-12 |

## Модуль 0. Звуки и письмо (два подэтапа)

Усилен по бенчмарку: у Кварчелии на фонетику отведено **2 полных урока до основного курса** — при 56 согласных это оправдано. Разбиваем модуль 0 на два подэтапа.

**0A. Звуки и различение** (без нагрузки на письмо): артикуляция сложных согласных, минимальные пары, контрасты, теряемые русскоговорящими, ритм и ударение — всё на аудио и повторении за диктором.

**0B. Письмо и чтение**: алфавит (56 согл. + 6 гл.), правила чтения, связь звук↔буква, набор слов.

| Поле | Содержание |
|---|---|
| Результат | ученик различает ключевые звуки (0A), читает учебные примеры и понимает устройство письма (0B) |
| Функции | услышать и различить звук, повторить за диктором, прочитать слово, набрать слово, спросить "как читать?" |
| Фонетика | 0A: сложные согласные, артикуляция, минимальные пары, ритм, ударение. 0B: алфавит, правила чтения |
| Лексика | 150-200 визуально и аудиально частотных слов (с картинками) |
| Грамматика | только базовая идея: слово, фраза, вопрос, утверждение |
| Аудио | алфавит, минимальные пары, slow repetition; аудио — ведущий канал этого модуля |
| Контроль | 0A: аудио-различение звуков/минимальных пар; 0B: чтение 20 слов, аудио-узнавание 20 слов |

## Глава 1. Знакомство

| Поле | Содержание |
|---|---|
| Результат | ученик здоровается, прощается, называет имя, реагирует на простые формулы |
| Функции | приветствие, прощание, имя, вежливая реакция, "да/нет" |
| Фонетика | 1 сложный звук/контраст из диалога |
| Лексика | 40-60 слов и формул |
| Грамматика | личные местоимения; **грамматический класс: человек (одуш.) vs вещь (неодуш.)** и **мужская/женская форма 2-го лица** (обязательный фундамент — без него не строятся «как тебя зовут?» и «я вижу X»); простое именное высказывание; вопросительная интонация. Класс объясняем через контраст с русским родом (блок ошибок русскоговорящих). *(Из бенчмарка Кварчелии: это поднимается в самый первый содержательный урок.)* |
| Диалог | 2 коротких диалога: нейтральный и чуть более вежливый; в т.ч. обращение к мужчине и к женщине |
| Упражнения | выбрать реплику, собрать диалог, подставить имя, прослушать и отметить |
| Контроль | ученик ведет 30-секундный диалог знакомства |

## Глава 2. Я, страна, язык

| Поле | Содержание |
|---|---|
| Результат | ученик говорит, кто он, откуда, какие языки знает/учит |
| Функции | "я из...", "я говорю...", "я учу...", "повторите, пожалуйста" |
| Фонетика | ритм короткой фразы |
| Лексика | страны, языки, базовые учебные команды |
| Грамматика | личные местоимения, простое отрицание, вопросы о происхождении и языке |
| Диалог | знакомство на языковом курсе |
| Упражнения | карточки стран/языков, мини-анкета, аудио-диктант |
| Контроль | ученик заполняет и озвучивает короткую анкету |

## Глава 3. Семья и люди

| Поле | Содержание |
|---|---|
| Результат | ученик рассказывает о 2-4 членах семьи и описывает человека |
| Функции | "это мой/моя...", возраст, профессия, простое описание |
| Фонетика | контраст, часто теряемый русскоговорящими |
| Лексика | родство, люди, профессии, числа 1-20 |
| Грамматика | притяжательность, указательные модели, числительные |
| Диалог | разговор о семье по фотографии |
| Упражнения | семейная схема, описание картинки, подстановка чисел |
| Контроль | ученик делает 5 предложений о семье |

## Глава 4. День и действия

| Поле | Содержание |
|---|---|
| Результат | ученик говорит о простых ежедневных действиях |
| Функции | "я иду/делаю/ем/читаю", распорядок дня, вопрос "что ты делаешь?" |
| Фонетика | ритм глагольной формы |
| Лексика | 20-30 частотных действий, части дня |
| Грамматика | первые глагольные модели, участники действия, настоящее время на ограниченном наборе |
| Диалог | короткий разговор о дне |
| Упражнения | pattern drills, замена субъекта/объекта, ответы по картинке |
| Контроль | ученик рассказывает 5-7 фраз о своем дне |

## Глава 5. Дом и город

| Поле | Содержание |
|---|---|
| Результат | ученик спрашивает и объясняет, где находится место/предмет, называет положение вещей в комнате |
| Функции | где? куда? откуда? около/без; «на стене», «на столе», «за домом» |
| Фонетика | лабиализованные согласные в частотных бытовых словах |
| Лексика | дом, комната, мебель, улица, школа, магазин, транспорт, ориентиры, цвета |
| Грамматика | **послелоги/локативы после существительного** (аҵы «в», ахь «в/куда», аҟынтәи «из», авара «около», ада «без» — Кварчелия, урок 9); **позиции** игылоуп/иҟәоуп/икнаҳауп/иҟәгылоуп (стоит/лежит/висит/сидит) с согласованием по классу. Контраст: у русского предлог впереди, у абхаза — сзади (см. `RUSSIAN_SPEAKER_ERRORS_REGISTER`, §4) |
| Диалог | «как пройти?» и описание своей комнаты |
| Упражнения | картиночные схемы места/направления, подписи к комнате, финал — двусторонний перевод «на стене/на столе/из дома…» |
| Контроль | ученик объясняет маршрут на простой карте и описывает комнату 5-7 фразами |

## Глава 6. Еда и покупки

| Поле | Содержание |
|---|---|
| Результат | ученик покупает простые вещи и говорит о еде |
| Функции | попросить, выбрать, спросить цену, назвать количество, поблагодарить |
| Фонетика | абруптивные согласные в названиях еды |
| Лексика | еда, напитки, магазин, числа, единицы количества |
| Грамматика | просьбы (вежливая форма), количество/числительные, объект в простом высказывании |
| Диалог | сцена в магазине/на рынке |
| Упражнения | карточки товаров и цен, ролевые реплики, финал — двусторонний перевод |
| Контроль | ролевая сцена "магазин" |

## Глава 7. Время и погода

| Поле | Содержание |
|---|---|
| Результат | ученик говорит о времени, возрасте, погоде и простых планах; здесь начинается сквозной слой «Повторение» |
| Функции | сегодня/завтра, дни недели, часы, «сколько лет?», даты/годы, погода, состояние |
| Фонетика | ритм числительных и дат |
| Лексика | календарь, части дня, погода, числа словами (1-20), состояние (холодно/тепло/хочу спать) |
| Грамматика | временные слова; **возраст** (Шаҟа шықәса ухыҵуеи? — Кварчелия, урок 9); **даты/годы словами**; простые плановые конструкции |
| Диалог | «сколько тебе лет?» и план на день |
| Упражнения | заполнение календаря, запись дат словами, аудио-диктант, финал — двусторонний перевод |
| Контроль | ученик рассказывает план на день и называет свой возраст и дату |

## Глава 8. Помощь и сервис

| Поле | Содержание |
|---|---|
| Результат | ученик просит помощь в бытовой ситуации и пользуется вежливыми формулами |
| Функции | помощь, телефон, документы, дорога, уточнение; спасибо/извините/приятного аппетита |
| Фонетика | интонация вопроса и просьбы |
| Лексика | сервисные ситуации, просьбы, проблемы, бытовые формулы (иҭабуп, саҭамыз — Кварчелия, урок 9) |
| Грамматика | **образование вопроса из утверждения** (формально, не только интонацией — см. `RUSSIAN_SPEAKER_ERRORS_REGISTER`, §6); вопросительные и модальные модели; вежливая просьба |
| Диалог | 3 сервисные мини-сцены |
| Упражнения | превращение утверждений в вопросы, подбор формул к ситуации, финал — двусторонний перевод |
| Контроль | ученик решает 3 сервисные мини-ситуации |

## Глава 10. Культура и чтение

| Поле | Содержание |
|---|---|
| Результат | ученик читает короткий адаптированный культурный текст |
| Функции | понять основную мысль, ответить на вопросы, пересказать |
| Фонетика | чтение вслух связного текста, интонация |
| Лексика | культурные реалии, пословицы, фольклорные мотивы (адаптированно, современно поданные — не «колхозный» пласт, см. `METHODOLOGY_IMPROVEMENTS`) |
| Грамматика | повторение и чтение реальных примеров |
| Диалог | обсуждение прочитанного |
| Упражнения | вопросы к тексту, восстановление пропусков, пересказ по опорам |

## Глава 11. Вариативность

| Поле | Содержание |
|---|---|
| Результат | ученик понимает, что такое литературная норма и почему встречаются варианты |
| Функции | распознать норму/вариант/диалектное |
| Фонетика | сопоставление норма/диалект на слух (абжуйская база vs варианты) |
| Лексика | метаязык: норма, вариант, диалект, разговорная речь |
| Грамматика | без новых тем, только сопоставление |
| Диалог | одна и та же фраза в норме и в разговорном варианте |
| Упражнения | распознать помету, отличить норму от варианта, соотнести с картой диалектов |
| Контроль | ученик правильно читает пометы в словаре/уроке |

## Глава 12. Итоговый модуль

| Поле | Содержание |
|---|---|
| Результат | ученик проходит интегральный контроль A1-A2 |
| Функции | диалог, монолог, чтение, аудирование, письмо |
| Контроль | тест, speaking prompts, listening prompts, short writing |

## Сквозные элементы

- Каждая глава строится по триаде «ввод → отработка → двусторонний перевод» (см. выше).
- Каждая глава возвращает 20-30% материала прошлых глав.
- **Сквозной слой «Повторение»** начиная с главы 7: отдельные упражнения на закрепление пройденного (по образцу раздела «Повторение» с 15-го урока у Кварчелии).
- Ядро лексики подаётся **визуально** (картинка + слово + аудио), не только списком.
- Каждая глава имеет 1 фонетический фокус.
- Каждая глава имеет slow и natural audio.
- Тесты — отдельным слоем контроля (образец формата — «100 тестов» Кварчелии, к анализу).
- Каждая глава имеет `source_id` и `rights_status`.
- Все абхазские фразы до проверки имеют статус `needs_native_review`.

## Приёмочные критерии главы (из анализа жалоб носителей, `METHODOLOGY_IMPROVEMENTS.md`)

Существующие абхазские учебники критикуют за то, что они рассчитаны на носителей и учат «на языке», а не «языку», плюс устаревшее содержание (колхозы, пахота). Наш контрпринцип — каждая глава проходит два теста:

1. **Понятно новичку:** материал понятен человеку, открывшему абхазский впервые, без учителя (объяснение по-русски + ключи + аудио).
2. **Естественно носителю:** каждая фраза проходит native review на естественность.
3. **Современное содержание:** быт, город, реальные ситуации; устаревших реалий избегаем, фольклор — позже (A2+) и адаптированно.
4. **Контрастив:** сквозной блок «Ошибки русскоговорящих» — трудное явление объясняется через сравнение с русским. Регистр явлений с привязкой к главам: `05_methodology/RUSSIAN_SPEAKER_ERRORS_REGISTER.md` (класс vs род, муж./жен. 2 л., притяжательные префиксы, послелоги, абруптивные согласные, да/нет).
